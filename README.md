# HIMATKN Website

Website resmi Himpunan Mahasiswa Teknik Nuklir (HIMATKN) yang dibangun menggunakan Flask dan MySQL.

## Fitur

- Dashboard Admin
- Manajemen Berita
- Manajemen Program Kerja
- Manajemen Pengurus
- REST API
- Tracking Pengunjung

## Teknologi

- Python
- Flask
- MySQL
- Jinja2
- HTML/CSS/JavaScript
